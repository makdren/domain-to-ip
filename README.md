# 🚀 Domain to IP — Массовый резолвер доменов в IP

**Быстрый и удобный инструмент** для преобразования списка доменов в IP-адреса.

Поддерживает **любой формат** записи доменов в файле.  
При каждом запуске **старые файлы `ips.txt` и `domain_ip.txt` автоматически удаляются** и создаются заново — никаких устаревших данных.

---

## ✨ Основные возможности

- Максимальная скорость — **50 потоков** одновременно
- Красивый прогресс-бар в реальном времени
- При каждом запуске **полностью удаляются** старые `ips.txt` и `domain_ip.txt`
- Сохранение **исходного порядка** доменов
- Нормализация входных записей: можно указывать домены как `example.com`, так и URL вида `https://example.com/path`
- Автоматическая фильтрация невалидных записей с понятным отчётом в консоли
- Повторные DNS-попытки при временных сбоях сети (настраивается через CLI)
- Подробный вывод ошибок
- Никаких внешних библиотек — только стандартный Python

## 📥 Установка

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/makdren/domain-to-ip/main/install.sh)"
```

Скрипт `install.sh`:
- Создаст папку `dom2ip`
- Скачает `dom2ip.py`
- Установит Python3 (если его нет)
- Покажет готовую инструкцию к запуску

---

## 🚀 Быстрый старт

### Заходим в папку скрипта
```bash
cd dom2ip
```
### Добавляете домены в domains.txt
```bash
nano domains.txt
```
### Запуск скрипта
```bash
python3 dom2ip.py
```

---

## 📝 Как добавить домены

Открой файл `domains.txt` и добавь домены **в любом удобном виде**:

1. В столбик
```txt
example.com
google.com
```
2. Через запятую и пробел
```txt
yandex.ru, mail.ru, github.com
```
3. Через пробел
```txt
gitlab.com bitbucket.org openai.com
```
4. Полные URL (скрипт сам выделит домен)
```txt
https://openai.com/research
http://github.com:80/features
```

---

## 📁 Что будет после запуска

Скрипт создаст два файла **с нуля**:

| Файл            | Содержимое                          | Назначение                     |
|-----------------|-------------------------------------|--------------------------------|
| `ips.txt`       | Только IP-адреса (по одному на строку) | Для iptables, nginx, haproxy и т.д. |
| `domain_ip.txt` | `домен   IP`                        | Удобно для отладки и хранения |

**Важно:** При каждом новом запуске старые версии этих файлов **удаляются автоматически**.

## ⚙️ Настройки (по желанию)

Через аргументы командной строки:

```bash
python3 dom2ip.py \
  --domains domains.txt \
  --threads 50 \
  --timeout 8 \
  --retries 2 \
  --retry-delay 0.25
```

- `--domains` — путь к файлу с доменами
- `--threads` — количество параллельных потоков
- `--timeout` — таймаут DNS-запроса (в секундах)
- `--retries` — количество попыток DNS-резолва для каждого домена
- `--retry-delay` — пауза между попытками (в секундах)

## 📌 Примечания

- Работает на **Python 3.6+**
- Использует системный DNS-резолвер (`socket.gethostbyname`)
- Поддерживает кириллические домены (IDN)
- При повторном запуске всегда выдаёт **актуальные** IP-адреса
